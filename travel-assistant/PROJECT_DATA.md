## 4. OPEN SOURCE DATA SOURCES (No API Key Required)

### Weather Data
- **Open-Meteo API**: https://api.open-meteo.com/v1/forecast
  - Free, no API key, REST-based
  - Returns hourly/daily temp, precipitation, wind, weathercode
  - Rate limit: 10,000/day free

### Geocoding
- **OpenStreetMap Nominatim**: https://nominatim.openstreetmap.org/search
  - Free, no API key
  - Convert city name → lat/lon
  - Rate limit: 1 req/sec (fair use)

### Points of Interest
- **Overpass API**: https://overpass-api.de/api/interpreter
  - Free OpenStreetMap data
  - Query hotels, restaurants, attractions near coordinates
  - No API key needed

### Wikipedia Info
- **Wikipedia REST API**: https://en.wikipedia.org/api/rest_v1/page/summary/{title}
  - Free, no API key
  - Returns description, extract, thumbnail for any topic

### Disaster/Safety Alerts
- **GDACS RSS Feed**: https://gdacs.org/xml/rss.xml
  - Global Disaster Alert and Coordination System
  - Free RSS, parseable with Rome/Jackson
  - Covers earthquakes, floods, cyclones, volcanoes

### Maps
- **Leaflet.js + OpenStreetMap tiles**
  - 100% free and open source
  - No API key needed for OSM tiles

### Static Images
- **Unsplash Source**: https://source.unsplash.com/800x600/?{keyword}
  - Free, no API key for basic usage
  - Used for destination photos

## 5. BUDGET CALCULATION FORMULA

```
Total Trip Cost = (Transport Cost) + (Accommodation Cost) + (Food Cost) + (Activities Cost) + (Miscellaneous 10%)

Per Day breakdown:
LOW:     Accommodation ₹800 + Food ₹500 + Activities ₹300 + Misc ₹200   = ₹1,800–2,000/person/day
MID:     Accommodation ₹3,000 + Food ₹1,200 + Activities ₹800 + Misc ₹500 = ₹5,500/person/day  
LUXURY:  Accommodation ₹12,000 + Food ₹4,000 + Activities ₹3,000 + Misc ₹2,000 = ₹21,000/person/day

Transport Add-on:
FLIGHT: +₹3,000–₹10,000 (one way)
TRAIN:  +₹300–₹3,000 (based on class)
BUS:    +₹200–₹1,500
CAB:    +₹15–₹25 per km
```

## 6. WEATHER SCORING ALGORITHM

```
Score (1–10) based on:
- Temperature comfort (20–32°C = best)
- Rainfall probability (lower = better)
- Humidity (lower = better)
- UV index (moderate = best)

Season Classification:
PEAK    = Score 8–10 (best time to visit)
GOOD    = Score 6–7 (acceptable)
AVOID   = Score 1–5 (monsoon/extreme heat/cold)
```

## 7. AI RECOMMENDATION RULES (Rule-Based Engine)

```
IF budget = LOW AND distance > 500km → recommend TRAIN (cheapest)
IF budget = LOW AND distance < 300km → recommend BUS
IF budget = MID AND duration_days >= 5 → recommend FLIGHT (save time)
IF budget = LUXURY → always recommend FLIGHT + private cab
IF destination_type = BEACH → warn about monsoon months
IF destination_type = MOUNTAIN → warn about landslide/snow months
IF people >= 4 → suggest GROUP booking discount
IF people = 2 → suggest honeymoon packages
IF duration_days = 1 → suggest NEARBY destinations only
IF weather_score < 5 → show disaster alert warning
```
